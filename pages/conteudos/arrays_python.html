1. O que é um Array (Lista) em Python?

Em Python, o equivalente ao array é chamado de list (lista). Uma lista é uma estrutura de dados que armazena vários valores em uma única variável, organizados em sequência e acessados por índice.

💡 Uma lista pode guardar números, textos, booleanos — e até outras listas! Os elementos ficam entre colchetes [ ] e separados por vírgulas.

1.1 Criando listas

# Lista de números
notas = [7.5, 8.0, 9.5, 6.0, 10.0]

# Lista de textos
frutas = ['maçã', 'banana', 'laranja', 'uva']

# Lista mista
dados = ['João', 20, True, 9.5]

# Lista vazia (para preencher depois)
vazia = []

1.2 Acessando elementos pelo índice

Cada elemento da lista possui um índice (posição), que começa em 0. O último elemento tem índice -1.

Valor 'maçã' 'banana' 'laranja' 'uva'
Índice+ 0 1 2 3
Índice- -4 -3 -2 -1 
frutas = ['maçã', 'banana', 'laranja', 'uva']

print(frutas[0])   # maçã   (primeiro elemento)
print(frutas[2])   # laranja
print(frutas[-1])  # uva    (último elemento)
print(frutas[-2])  # laranja

2. Percorrendo uma lista com for

O laço for é a forma mais comum de percorrer todos os elementos de uma lista, um por um, sem precisar usar índices.

2.1 for direto nos elementos

frutas = ['maçã', 'banana', 'laranja', 'uva']

for fruta in frutas:
    print(fruta)

# Saída:
# maçã
# banana
# laranja
# uva

2.2 for com índice usando enumerate()

Quando você precisa do índice e do valor ao mesmo tempo, use enumerate():

frutas = ['maçã', 'banana', 'laranja', 'uva']

for indice, fruta in enumerate(frutas):
    print(indice, '-', fruta)

# Saída:
# 0 - maçã
# 1 - banana
# 2 - laranja
# 3 - uva

2.3 for com range() pelo índice

notas = [7.5, 8.0, 9.5, 6.0]

for i in range(len(notas)):
    print(f'Aluno {i+1}: nota {notas[i]}')

# Saída:
# Aluno 1: nota 7.5
# Aluno 2: nota 8.0
# Aluno 3: nota 9.5
# Aluno 4: nota 6.0

3. Funções essenciais das listas

3.1 len()

Retorna o número total de elementos da lista.
Sintaxe len(lista)
Exemplo len([7, 8, 9])
Saída 3 
notas = [7.5, 8.0, 9.5, 6.0, 10.0]

tamanho = len(notas)
print('A lista tem', tamanho, 'elementos')
# A lista tem 5 elementos

3.2 append()

Adiciona um elemento ao final da lista.
Sintaxe lista.append(elemento)
Exemplo frutas.append('banana')
Saída ['maçã', 'banana'] 
compras = ['arroz', 'feijão']

compras.append('macarrão')
compras.append('sal')

print(compras)
# ['arroz', 'feijão', 'macarrão', 'sal']

3.3 insert()

Insere um elemento em uma posição específica da lista.
Sintaxe lista.insert(indice, elemento)
Exemplo frutas.insert(1, 'banana')
Saída ['maçã', 'banana', 'laranja'] 
times = ['Flamengo', 'Palmeiras', 'Santos']

# Insere na posição 1 (segundo lugar)
times.insert(1, 'Corinthians')

print(times)
# ['Flamengo', 'Corinthians', 'Palmeiras', 'Santos']

3.4 pop()

Remove e retorna o elemento de uma posição (padrão: último).
Sintaxe lista.pop() ou lista.pop(indice)
Exemplo nums = [1, 2, 3, 4]; nums.pop()
Saída removido = 4 / nums = [1, 2, 3] 
fila = ['Ana', 'Bruno', 'Carlos', 'Diana']

# Remove o último
ultimo = fila.pop()
print('Removido:', ultimo)   # Diana
print(fila)                  # ['Ana', 'Bruno', 'Carlos']

# Remove pelo índice
primeiro = fila.pop(0)
print('Removido:', primeiro) # Ana
print(fila)                  # ['Bruno', 'Carlos']

3.5 remove()

Remove a primeira ocorrência de um valor específico.
Sintaxe lista.remove(valor)
Exemplo cores = ['azul', 'verde', 'azul']; cores.remove('azul')
Saída ['verde', 'azul'] 
notas = [7, 5, 9, 5, 8]

notas.remove(5)   # remove o PRIMEIRO 5

print(notas)
# [7, 9, 5, 8]

⚠️ Atenção: se o valor não existir na lista, o Python levanta um ValueError. Use in para verificar antes de remover.

if 5 in notas:
    notas.remove(5)

3.6 index()

Retorna o índice (posição) da primeira ocorrência de um valor.
Sintaxe lista.index(valor)
Exemplo frutas.index('banana')
Saída 1 
alunos = ['Ana', 'Bruno', 'Carlos', 'Diana']

pos = alunos.index('Carlos')
print('Carlos está na posição:', pos)
# Carlos está na posição: 2

3.7 reverse()

Inverte a ordem dos elementos na própria lista (modifica diretamente).
Sintaxe lista.reverse()
Exemplo nums = [1, 2, 3, 4, 5]; nums.reverse()
Saída [5, 4, 3, 2, 1] 
palavras = ['Python', 'é', 'muito', 'legal']

palavras.reverse()

print(palavras)
# ['legal', 'muito', 'é', 'Python']

💡 Dica: reverse() altera a lista original. Se quiser manter a original, use lista[::-1], que cria uma cópia invertida.

3.8 count()

Conta quantas vezes um valor aparece na lista.
Sintaxe lista.count(valor)
Exemplo notas = [7, 8, 7, 9, 7]; notas.count(7)
Saída 3 
respostas = ['A', 'B', 'A', 'C', 'A', 'B', 'A']

qtd_A = respostas.count('A')
qtd_B = respostas.count('B')

print('Resposta A:', qtd_A, 'vezes')  # 4 vezes
print('Resposta B:', qtd_B, 'vezes')  # 2 vezes

3.9 copy()

Cria uma cópia independente da lista (sem copiar o endereço de memória).
Sintaxe nova = lista.copy()
Exemplo b = a.copy(); b.append(4)
Saída a = [1, 2, 3] / b = [1, 2, 3, 4] 
original = [1, 2, 3, 4, 5]

# SEM copy — ambas apontam para o mesmo lugar!
errada = original
errada.append(99)
print(original)  # [1, 2, 3, 4, 5, 99]  <- MODIFICOU!

# COM copy — listas independentes
certa = original.copy()
certa.append(100)
print(original)  # [1, 2, 3, 4, 5, 99]  <- intacta
print(certa)     # [1, 2, 3, 4, 5, 99, 100]

3.10 extend()

Adiciona todos os elementos de outra lista ao final da lista atual.
Sintaxe lista.extend(outra_lista)
Exemplo a = [1, 2, 3]; a.extend([4, 5, 6])
Saída [1, 2, 3, 4, 5, 6] 
turma_manha = ['Ana', 'Bruno', 'Carlos']
turma_tarde  = ['Diana', 'Eduardo', 'Fernanda']

turma_manha.extend(turma_tarde)

print(turma_manha)
# ['Ana', 'Bruno', 'Carlos', 'Diana', 'Eduardo', 'Fernanda']

💡 Diferença: append(['a','b']) adiciona a lista inteira como 1 elemento. extend(['a','b']) adiciona cada elemento separadamente.

4. Resumo — todas as funções

Função Sintaxe O que faz
len() len(lista) Retorna o tamanho da lista
append() lista.append(x) Adiciona x ao final
insert() lista.insert(i, x) Insere x na posição i
pop() lista.pop() / lista.pop(i) Remove e retorna o último (ou posição i)
remove() lista.remove(x) Remove a primeira ocorrência de x
index() lista.index(x) Retorna o índice de x
reverse() lista.reverse() Inverte a lista in-place
count() lista.count(x) Conta quantas vezes x aparece
copy() lista.copy() Retorna uma cópia independente
extend() lista.extend(outra) Adiciona todos os elementos de outra lista

Exercícios — Arrays (Listas) em Python

6 exercícios práticos com entrada de dados, do básico ao avançado.

Exercício 1 — Cadastro e exibição de notas

Descrição

Você foi contratado para criar um sistema simples de cadastro de notas para uma turma. O programa deve pedir ao usuário quantos alunos existem na turma e, em seguida, solicitar a nota de cada um. Ao final, o programa deve exibir todas as notas cadastradas, uma por linha, com o número do aluno.

Requisitos

  • Usar uma lista vazia e append() para adicionar as notas
  • Usar for com range() para solicitar as entradas
  • Exibir o resultado com for e enumerate()

Exemplo de execução

Quantos alunos na turma? 4

Digite a nota do aluno 1: 7.5
Digite a nota do aluno 2: 9.0
Digite a nota do aluno 3: 6.0
Digite a nota do aluno 4: 8.5

--- Notas cadastradas ---
Aluno 1: 7.5
Aluno 2: 9.0
Aluno 3: 6.0
Aluno 4: 8.5

Dicas

  • Use int(input(...)) para ler a quantidade de alunos
  • Use float(input(...)) para ler as notas, pois podem ter casas decimais
  • Lembre-se que enumerate() começa do índice 0 — some 1 para exibir o número correto do aluno

Exercício 2 — Maior, menor e média da turma

Descrição

Continuando o sistema de notas do exercício anterior, agora o programa deve calcular e exibir três informações sobre as notas cadastradas: a maior nota, a menor nota e a média da turma. O usuário informa a quantidade de alunos e suas respectivas notas, e o programa processa os dados.

Requisitos

  • Usar append() para montar a lista de notas
  • Calcular a média somando todos os elementos com um for e dividindo pelo total
  • Usar len() para obter o total de alunos
  • Encontrar o maior e o menor valor percorrendo a lista (sem usar max() ou min() — tente com if dentro do for)

Exemplo de execução

Quantos alunos na turma? 5

Digite a nota do aluno 1: 8.0
Digite a nota do aluno 2: 5.5
Digite a nota do aluno 3: 9.5
Digite a nota do aluno 4: 7.0
Digite a nota do aluno 5: 6.5

--- Resultado ---
Maior nota : 9.5
Menor nota : 5.5
Média      : 7.3

Dicas

  • Para encontrar o maior valor manualmente: crie uma variável maior começando com lista[0] e compare com cada elemento no for
  • Para a média: some todos os valores em uma variável soma e divida por len(lista) ao final
  • Use round(media, 1) para exibir apenas uma casa decimal

Exercício 3 — Fila de atendimento

Descrição

Uma clínica precisa de um sistema de gerenciamento de fila de atendimento. O programa deve primeiro cadastrar os nomes dos pacientes que chegaram. Depois, deve simular o atendimento: a cada vez que o usuário pressionar Enter, o primeiro da fila é atendido e removido, exibindo uma mensagem de atendimento. O processo continua até a fila estar vazia.

Requisitos

  • Usar append() para cadastrar os pacientes
  • Usar pop(0) para remover e retornar o primeiro da fila a cada atendimento
  • Usar len() para mostrar quantas pessoas ainda aguardam
  • Usar um while para continuar enquanto houver pacientes

Exemplo de execução

Quantos pacientes na fila? 3

Nome do paciente 1: Ana
Nome do paciente 2: Bruno
Nome do paciente 3: Carlos

--- Atendimento iniciado ---

Fila atual: ['Ana', 'Bruno', 'Carlos']
Pressione Enter para chamar o próximo...

Atendendo: Ana
Ainda aguardando: 2 paciente(s)

Fila atual: ['Bruno', 'Carlos']
Pressione Enter para chamar o próximo...

Atendendo: Bruno
Ainda aguardando: 1 paciente(s)

Fila atual: ['Carlos']
Pressione Enter para chamar o próximo...

Atendendo: Carlos

Fila encerrada. Todos os pacientes foram atendidos!

Dicas

  • Para “pressionar Enter”, use input() sem nenhum texto — ele apenas pausa até o usuário apertar Enter
  • pop(0) remove e retorna o primeiro elemento: paciente = fila.pop(0)
  • Verifique len(fila) > 0 antes de mostrar quantos ainda aguardam para evitar exibir 0 paciente(s) após o último

Exercício 4 — Removendo repetidos da lista

Descrição

Um sistema de cadastro recebeu uma lista de produtos digitados manualmente e, por descuido, vários nomes foram inseridos mais de uma vez. Seu programa deve receber os produtos um a um, verificar se o produto já foi cadastrado e, caso tenha sido, informar ao usuário que o item é duplicado e não adicioná-lo novamente. Ao final, exibe a lista limpa e sem repetições.

Requisitos

  • Usar in para verificar se o elemento já está na lista antes de adicionar
  • Usar append() apenas quando o item for novo
  • Usar count() ao final para confirmar que não há repetições
  • O usuário digita "fim" para encerrar o cadastro

Exemplo de execução

Digite os produtos (ou 'fim' para encerrar):

Produto: arroz
Produto: feijão
Produto: arroz
⚠️  'arroz' já está na lista. Ignorando.
Produto: macarrão
Produto: feijão
⚠️  'feijão' já está na lista. Ignorando.
Produto: sal
Produto: fim

--- Lista final de produtos ---
1. arroz
2. feijão
3. macarrão
4. sal

Total de produtos únicos: 4

Dicas

  • Use um while True com break quando o usuário digitar "fim"
  • Compare sempre em letras minúsculas para evitar que "Arroz" e "arroz" sejam tratados como diferentes: produto.lower()
  • Para listar com número, use enumerate(lista, start=1) que já começa pelo índice 1

Exercício 5 — Fusão e ordenação de duas turmas

Descrição

Duas turmas de uma escola fizeram a mesma prova. Cada turma tem sua própria lista de notas. O programa deve receber as notas das duas turmas separadamente, juntá-las em uma única lista usando extend(), calcular a média geral e exibir quantos alunos ficaram acima da média e quantos ficaram abaixo ou na média. Por fim, deve exibir a lista completa ordenada do maior para o menor usando reverse() após ordenar com sort().

Requisitos

  • Usar append() para cada turma separadamente
  • Usar extend() para unir as duas listas
  • Usar len() para contar o total de alunos
  • Usar sort() e reverse() para ordenar do maior para o menor
  • Usar count() não é necessário aqui — use if dentro de um for para contar acima/abaixo da média

Exemplo de execução

--- Turma A ---
Quantos alunos na Turma A? 3
Nota do aluno 1: 6.0
Nota do aluno 2: 8.5
Nota do aluno 3: 7.0

--- Turma B ---
Quantos alunos na Turma B? 3
Nota do aluno 1: 9.0
Nota do aluno 2: 5.5
Nota do aluno 3: 7.5

--- Resultado Geral ---
Total de alunos : 6
Média geral     : 7.25

Acima da média  : 3 aluno(s) → [8.5, 9.0, 7.5]
Abaixo/na média : 3 aluno(s) → [6.0, 7.0, 5.5]

--- Ranking (maior para menor) ---
1º lugar: 9.0
2º lugar: 8.5
3º lugar: 7.5
4º lugar: 7.0
5º lugar: 6.0
6º lugar: 5.5

Dicas

  • Monte as listas das turmas A e B separadamente, depois use turma_a.extend(turma_b) para criar a lista geral
  • Para ordenar do maior para o menor: lista.sort() ordena crescente → depois lista.reverse() inverte
  • Para o ranking, use enumerate(lista_ordenada, start=1) e formate com f'{i}º lugar: {nota}'
  • Separe os alunos acima da média em uma lista auxiliar com append() dentro do for para exibir no resultado

💡 Dica geral: antes de escrever o código, leia o enunciado com calma, identifique quais funções de lista serão usadas e esboce o passo a passo em comentários no código. Isso facilita muito!
