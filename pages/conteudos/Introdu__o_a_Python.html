Aula 01 — Introdução ao Python: variáveis, tipos de dados, print() e input()

Disciplina: Algoritmos e Lógica de Programação | Turma: 1º Ano Redes
Objetivo: Compreender o que é Python, como declarar variáveis, os principais tipos de dados e como interagir com o usuário via print() e input().

1. O que é Python?

Python é uma linguagem de programação de alto nível, criada por Guido van Rossum e lançada em 1991. É uma das linguagens mais populares do mundo, usada em áreas como:

  • Desenvolvimento web (Django, Flask)
  • Ciência de dados e inteligência artificial
  • Automação de tarefas
  • Segurança e redes
  • Jogos e aplicativos

Por que Python é uma boa primeira linguagem?

  • Sintaxe próxima do inglês — fácil de ler e escrever
  • Não precisa declarar tipos de variáveis explicitamente
  • Erros são claros e descritivos
  • Enorme comunidade e quantidade de recursos gratuitos

Seu primeiro programa

Abra o Vscode ou qualquer editor de Python e digite:

print("Olá, mundo!")

Saída:

Olá, mundo!

Parabéns — você acabou de escrever e executar seu primeiro programa em Python!

2. Comentários

Comentários são linhas ignoradas pelo Python — servem para explicar o código para quem vai ler.

# Isso é um comentário de linha única

# O Python ignora tudo que começa com #
print("Isso será executado")  # comentário no final da linha

"""
Isso é um comentário
de múltiplas linhas
(tecnicamente é uma string não atribuída)
"""

💡 Bons comentários explicam o “porquê”, não o “o quê”. O código já mostra o que está sendo feito; o comentário deve explicar a intenção.

3. Variáveis

Uma variável é um espaço na memória do computador onde guardamos um valor. Pense como uma caixa com uma etiqueta: a etiqueta é o nome da variável, e o conteúdo da caixa é o valor armazenado.

Criando variáveis

Em Python, basta escrever o nome, o sinal = e o valor:

nome = "Ana"
idade = 17
altura = 1.65
aprovado = True

⚠️ O = em programação não significa igualdade matemática — significa atribuição: “guarde este valor nesta variável”.

Regras para nomes de variáveis

Regra Exemplo válido Exemplo inválido
Letras, números e _ nota_1, total2 nota-1, total 2
Não pode começar com número media, _valor 1media, 3total
Case-sensitive (maiúsculas importam) nome ≠ Nome ≠ NOME —
Não usar palavras reservadas resultado if, for, return

Convenção: snake_case

Em Python, o padrão é usar snake_case para nomes de variáveis: todas as letras minúsculas, palavras separadas por underscore.

# ✅ Bom (snake_case)
nome_completo = "João Silva"
nota_final = 8.5
quantidade_alunos = 30

# ❌ Evitar
NomeCompleto = "João Silva"   # PascalCase (usado para classes)
notaFinal = 8.5               # camelCase (usado em outras linguagens)

Variáveis podem ser reatribuídas

pontuacao = 0
print(pontuacao)   # 0

pontuacao = 10
print(pontuacao)   # 10

pontuacao = pontuacao + 5
print(pontuacao)   # 15

4. Tipos de dados

Cada valor em Python tem um tipo. O tipo define quais operações podem ser feitas com aquele valor. Os principais tipos são:

4.1 int — Números inteiros

Números sem parte decimal — positivos, negativos ou zero.

ano = 2025
temperatura = -3
quantidade = 0
populacao = 215_000_000   # underscore como separador de milhar (só visual)

print(type(ano))   # <class 'int'>

Operações com inteiros:

a = 10
b = 3

print(a + b)   # 13  → soma
print(a - b)   # 7   → subtração
print(a * b)   # 30  → multiplicação
print(a / b)   # 3.333... → divisão (resultado sempre float!)
print(a // b)  # 3   → divisão inteira (descarta o decimal)
print(a % b)   # 1   → módulo (resto da divisão)
print(a ** b)  # 1000 → potenciação (10³)

4.2 float — Números decimais

Números com parte decimal (ponto flutuante). Use ponto, não vírgula.

altura = 1.75
preco = 29.90
pi = 3.14159
temperatura = -7.5

print(type(altura))   # <class 'float'>

⚠️ Em Python (e na maioria das linguagens), o separador decimal é o ponto (.), não a vírgula. 1,75 seria interpretado como dois valores separados por vírgula — causaria erro ou comportamento inesperado.

# Operações com float:
print(1.5 + 2.3)    # 3.8
print(10 / 3)       # 3.3333333333333335 (imprecisão de ponto flutuante — normal!)
print(round(10/3, 2))  # 3.33 → arredondando para 2 casas

4.3 str — Texto (string)

Qualquer sequência de caracteres entre aspas simples '...' ou duplas "...".

nome = "Maria"
cidade = 'Recife'
mensagem = "Olá, tudo bem?"
numero_como_texto = "42"   # é uma string, não um int!

print(type(nome))   # <class 'str'>

Operações com strings:

# Concatenação (juntar strings)
primeiro = "João"
ultimo = "Silva"
nome_completo = primeiro + " " + ultimo
print(nome_completo)   # João Silva

# Repetição
print("ha" * 3)   # hahaha

# Tamanho
print(len("Python"))   # 6

# Acesso por índice (começa em 0)
palavra = "Python"
print(palavra[0])   # P
print(palavra[1])   # y
print(palavra[-1])  # n  (último caractere)

f-strings (formatação moderna):

nome = "Ana"
idade = 17
nota = 8.756

# f-string: coloque f antes das aspas e use {} para variáveis
print(f"Aluna: {nome}")
print(f"Idade: {idade} anos")
print(f"Nota: {nota:.2f}")   # .2f = 2 casas decimais → 8.76

4.4 bool — Booleano (verdadeiro ou falso)

Representa apenas dois valores possíveis: True ou False. Muito usado em condições.

aprovado = True
reprovado = False
maior_de_idade = True

print(type(aprovado))   # <class 'bool'>

Comparações retornam booleano:

print(10 > 5)     # True
print(10 < 5)     # False
print(10 == 10)   # True  (== compara, = atribui!)
print(10 != 5)    # True  (diferente de)
print(10 >= 10)   # True

4.5 Tabela resumo dos tipos

Tipo Nome Exemplo type()
Inteiro int 42, -7, 0 <class 'int'>
Decimal float 3.14, -1.5 <class 'float'>
Texto str "Olá", 'Python' <class 'str'>
Booleano bool True, False <class 'bool'>

5. Conversão de tipos (casting)

Às vezes é necessário converter um tipo em outro. Python oferece funções para isso:

# String para inteiro
idade_texto = "17"
idade_numero = int(idade_texto)
print(idade_numero + 1)   # 18

# String para float
nota_texto = "8.5"
nota_numero = float(nota_texto)
print(nota_numero * 2)    # 17.0

# Número para string
valor = 42
valor_texto = str(valor)
print("O valor é: " + valor_texto)   # O valor é: 42

# Inteiro para float e vice-versa
print(float(5))    # 5.0
print(int(3.9))    # 3  (trunca, não arredonda!)

⚠️ Cuidado: tentar converter texto não numérico para número causa erro:

int("abc")   # ValueError: invalid literal for int()

6. A função print()

print() exibe informações na tela. É a principal ferramenta para mostrar resultados ao usuário.

Formas de usar:

# Exibindo valores simples
print("Olá!")
print(42)
print(3.14)
print(True)

# Exibindo variáveis
nome = "Carlos"
print(nome)

# Exibindo múltiplos valores (separados por vírgula)
print("Nome:", nome, "| Idade:", 17)
# saída: Nome: Carlos | Idade: 17

Parâmetros do print()

# sep: define o separador entre os valores (padrão é espaço)
print("a", "b", "c", sep="-")       # a-b-c
print("a", "b", "c", sep=" | ")     # a | b | c
print("a", "b", "c", sep="")        # abc

# end: define o que vem no final (padrão é \n — quebra de linha)
print("Linha 1", end=" ")
print("Linha 2")
# saída: Linha 1 Linha 2  (na mesma linha!)

# \n dentro de strings: quebra de linha manual
print("Linha 1\nLinha 2\nLinha 3")

f-strings com print() (forma mais moderna):

nome = "Ana"
nota = 9.333

print(f"Aluna: {nome}")
print(f"Nota final: {nota:.1f}")   # 9.3
print(f"Aprovada: {nota >= 7}")    # True

7. A função input()

input() pausa o programa e aguarda o usuário digitar algo. O valor digitado é sempre retornado como string.

Uso básico:

nome = input("Digite seu nome: ")
print(f"Olá, {nome}!")

Execução:

Digite seu nome: Maria
Olá, Maria!

input() sempre retorna string!

Este é o erro mais comum com iniciantes. Se você precisa de um número, converta com int() ou float():

# ❌ Errado: tenta somar string com número
idade = input("Sua idade: ")
print(idade + 1)   # TypeError!

# ✅ Correto: converte antes de operar
idade = int(input("Sua idade: "))
print(idade + 1)   # funciona!

# Para números decimais:
altura = float(input("Sua altura (ex: 1.75): "))
print(f"Altura informada: {altura} m")

Exemplo completo com input():

print("=== Calculadora de média ===")

nota1 = float(input("Digite a nota 1: "))
nota2 = float(input("Digite a nota 2: "))
nota3 = float(input("Digite a nota 3: "))

media = (nota1 + nota2 + nota3) / 3

print(f"\nNotas: {nota1}, {nota2}, {nota3}")
print(f"Média: {media:.2f}")

Execução:

=== Calculadora de média ===
Digite a nota 1: 7.5
Digite a nota 2: 8.0
Digite a nota 3: 9.0

Notas: 7.5, 8.0, 9.0
Média: 8.17

8. Resumo geral

# Variáveis e tipos
nome = "João"          # str
idade = 17             # int
altura = 1.72          # float
aprovado = True        # bool

# Verificando o tipo
print(type(nome))      # <class 'str'>

# Conversão
numero = int("42")     # str → int
texto = str(100)       # int → str

# Saída formatada
print(f"Aluno: {nome}, {idade} anos, altura: {altura:.2f}m")

# Entrada do usuário
turma = input("Qual sua turma? ")
print(f"Bem-vindo, aluno da turma {turma}!")

9. Exercícios

Instruções: teste cada programa no IDLE ou VS Code. Leia as dicas apenas se travar por mais de 5 minutos.

Exercício 1 — Apresentação pessoal 🟢 Fácil

Enunciado:

Crie um programa que use input() para pedir ao usuário seu nome, idade e cidade, e depois exiba uma mensagem de apresentação formatada.

Saída esperada:

Digite seu nome: Carlos
Digite sua idade: 17
Digite sua cidade: Caruaru

=== Apresentação ===
Olá! Me chamo Carlos, tenho 17 anos e moro em Caruaru.

Estrutura sugerida:

nome = input("Digite seu nome: ")
idade = input("Digite sua idade: ")
cidade = input("Digite sua cidade: ")

print("\n=== Apresentação ===")
# exiba a mensagem formatada aqui
💡 Dica

Use uma f-string:

print(f"Olá! Me chamo {nome}, tenho {idade} anos e moro em {cidade}.")

A idade pode ficar como string aqui, já que não vamos fazer cálculos com ela.

Exercício 2 — Calculadora de IMC 🟢 Fácil

Enunciado:

Peça ao usuário seu peso (em kg) e altura (em metros). Calcule o IMC usando a fórmula IMC = peso / altura² e exiba o resultado com 2 casas decimais.

Saída esperada:

Digite seu peso (kg): 70
Digite sua altura (m): 1.75

Seu IMC é: 22.86

Estrutura sugerida:

peso = float(input("Digite seu peso (kg): "))
altura = float(input("Digite sua altura (m): "))

imc = peso / altura ** 2

print(f"\nSeu IMC é: {imc:.2f}")
💡 Dica
  • float(input(...)) converte a entrada direto para número decimal.
  • altura ** 2 calcula altura ao quadrado.
  • {imc:.2f} formata o número com 2 casas decimais.

Exercício 3 — Tipos de dados na prática 🟢 Fácil

Enunciado:

Crie um programa que declare uma variável de cada tipo (int, float, str, bool), exiba os valores e o tipo de cada uma usando type(). Escolha variáveis que façam sentido juntas (ex: dados de um produto, aluno ou pessoa).

Saída esperada (exemplo com aluno):

=== Dados do aluno ===
Nome: Ana Souza     | Tipo: <class 'str'>
Matrícula: 2025001  | Tipo: <class 'int'>
Média: 8.75         | Tipo: <class 'float'>
Aprovado: True      | Tipo: <class 'bool'>

Estrutura sugerida:

nome = "Ana Souza"
matricula = 2025001
media = 8.75
aprovado = True

print("=== Dados do aluno ===")
print(f"Nome: {nome}     | Tipo: {type(nome)}")
# continue para as outras variáveis...
💡 Dica

type(variavel) retorna o tipo da variável. Você pode usá-lo diretamente dentro de uma f-string: f"{type(nome)}".

Exercício 4 — Conversor de temperatura 🟡 Médio

Enunciado:

Crie um programa que leia uma temperatura em Celsius e converta para Fahrenheit e Kelvin. Exiba os três valores com 1 casa decimal.

Fórmulas:

  • Fahrenheit: F = C × 1.8 + 32
  • Kelvin: K = C + 273.15

Saída esperada:

Digite a temperatura em Celsius: 100

=== Conversão de temperatura ===
Celsius:    100.0 °C
Fahrenheit: 212.0 °F
Kelvin:     373.2 K

Estrutura sugerida:

celsius = float(input("Digite a temperatura em Celsius: "))

fahrenheit = celsius * 1.8 + 32
kelvin = celsius + 273.15

print("\n=== Conversão de temperatura ===")
print(f"Celsius:    {celsius:.1f} °C")
# complete para fahrenheit e kelvin
💡 Dica

Use {valor:.1f} para formatar com 1 casa decimal. Basta substituir o nome da variável em cada linha do print().
