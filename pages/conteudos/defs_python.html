Funções em Python — def()

Disciplina: Programação Aplicada | Turma: 2º Ano Redes
Objetivo: Compreender o conceito de funções, sua sintaxe, parâmetros, retorno de valores e escopo de variáveis.

1. O que é uma função?

Uma função é um bloco de código nomeado que realiza uma tarefa específica. Você escreve esse bloco uma vez e pode executá-lo quantas vezes quiser, simplesmente chamando pelo seu nome.

Pense em funções como “receitas”: você escreve a receita do bolo uma vez, e pode fazer o bolo quantas vezes quiser sem precisar reescrever as instruções.

Por que usar funções?

Vantagem Explicação
Evita repetição Escreva o código uma vez, use várias vezes (princípio DRY — Don’t Repeat Yourself)
Organização Divide o programa em partes menores e mais fáceis de entender
Manutenção Para corrigir um erro, você muda em um só lugar
Reutilização A mesma função pode ser usada em projetos diferentes

2. Sintaxe básica

def nome_da_funcao(parametros):
    # corpo da função (indentado com 4 espaços)
    # instruções...
    return valor  # opcional

Explicando cada parte:

  • def → palavra-chave que inicia a definição da função
  • nome_da_funcao → identificador escolhido por você (use snake_case, ex: calcular_media)
  • (parametros) → dados de entrada que a função pode receber (pode ser vazio: ())
  • : → dois-pontos obrigatórios após o cabeçalho
  • corpo → bloco de código indentado (4 espaços ou 1 Tab)
  • return → devolve um valor ao ponto de chamada (opcional)

⚠️ Atenção: Em Python, a indentação não é opcional — ela faz parte da sintaxe. Todo o corpo da função precisa estar indentado.

3. Função sem parâmetros

A forma mais simples: a função executa sempre o mesmo comportamento, sem receber dados externos.

def saudacao():
    print("Olá! Bem-vindo à aula de Python!")

# Chamando a função:
saudacao()

Saída:

Olá! Bem-vindo à aula de Python!

Perceba que primeiro definimos a função com def, e depois a chamamos pelo nome seguido de parênteses. Sem os parênteses, Python apenas referencia a função sem executá-la.

4. Função com parâmetros

Parâmetros são variáveis que a função recebe como entrada. Eles permitem que a função trabalhe com dados diferentes a cada chamada.

def saudacao(nome):
    print(f"Olá, {nome}! Seja bem-vindo.")

# Chamando com argumentos diferentes:
saudacao("Ana")      # saída: Olá, Ana! Seja bem-vindo.
saudacao("Carlos")   # saída: Olá, Carlos! Seja bem-vindo.
saudacao("Turma")    # saída: Olá, Turma! Seja bem-vindo.

Terminologia importante:

  • Parâmetro → a variável no cabeçalho da função: def saudacao(nome)
  • Argumento → o valor passado na chamada: saudacao("Ana")

Função com múltiplos parâmetros

def apresentar(nome, idade, cidade):
    print(f"Nome: {nome}")
    print(f"Idade: {idade} anos")
    print(f"Cidade: {cidade}")

apresentar("Maria", 17, "Recife")

Saída:

Nome: Maria
Idade: 17 anos
Cidade: Recife

5. Retorno de valores com return

A instrução return faz a função devolver um resultado que pode ser armazenado em uma variável ou usado em expressões.

def somar(a, b):
    resultado = a + b
    return resultado

# O valor retornado pode ser guardado:
total = somar(10, 5)
print(total)          # saída: 15

# Ou usado diretamente em expressões:
print(somar(3, 7) * 2)  # saída: 20

Diferença entre print() e return

Esta é uma das confusões mais comuns entre iniciantes:

# Com print: exibe na tela, mas não retorna nada utilizável
def dobrar_print(x):
    print(x * 2)

# Com return: devolve o valor para quem chamou
def dobrar_return(x):
    return x * 2

# Testando a diferença:
a = dobrar_print(5)    # exibe 10 na tela
print(a)               # exibe: None  ← a função não retornou nada!

b = dobrar_return(5)   # não exibe nada
print(b)               # exibe: 10  ← o valor foi guardado em b

💡 Regra prática: Use return quando quiser usar o resultado da função em outras operações. Use print() apenas para exibir informação ao usuário.

6. Parâmetros com valor padrão (default)

Você pode definir um valor padrão para um parâmetro. Se o argumento não for passado na chamada, o valor padrão é usado automaticamente.

def potencia(base, expoente=2):
    return base ** expoente

print(potencia(3))      # 9  → usa expoente=2 (padrão)
print(potencia(3, 3))   # 27 → sobrescreve: usa expoente=3
print(potencia(2, 10))  # 1024

⚠️ Regra: Parâmetros com valor padrão devem vir depois dos parâmetros sem padrão.

def funcao(a, b=10):   # ✅ correto
    pass

def funcao(a=10, b):   # ❌ SyntaxError
    pass

7. Escopo de variáveis

Escopo define onde uma variável pode ser acessada no programa.

  • Variável local: criada dentro de uma função. Existe apenas enquanto a função executa.
  • Variável global: criada fora de qualquer função. Pode ser lida de dentro das funções.
x = 100  # variável global

def minha_funcao():
    y = 50  # variável local
    print(x)  # ✅ pode ler variável global: 100
    print(y)  # ✅ variável local: 50

minha_funcao()

print(x)  # ✅ funciona: 100
print(y)  # ❌ NameError: y não existe fora da função!

Visualizando o escopo:

┌─────────────────────────────────────┐
│  ESCOPO GLOBAL                      │
│  x = 100                            │
│                                     │
│  ┌────────────────────────────────┐ │
│  │  ESCOPO LOCAL (minha_funcao)   │ │
│  │  y = 50                        │ │
│  │  pode ler x ✅                 │ │
│  └────────────────────────────────┘ │
│                                     │
│  não pode ler y ❌                  │
└─────────────────────────────────────┘

💡 Boa prática: Evite depender de variáveis globais dentro das funções. Prefira passar os dados como parâmetros — isso torna o código mais previsível e fácil de testar.

8. Resumo da sintaxe

# Sem parâmetros, sem retorno
def mensagem():
    print("Olá!")

# Com parâmetros, sem retorno
def cumprimentar(nome):
    print(f"Olá, {nome}!")

# Com parâmetros e com retorno
def multiplicar(a, b):
    return a * b

# Com parâmetro padrão
def elevar(base, exp=2):
    return base ** exp

# Chamadas:
mensagem()
cumprimentar("João")
resultado = multiplicar(4, 5)
print(resultado)             # 20
print(elevar(3))             # 9
print(elevar(3, 4))          # 81

Exercício 4 — Maior de três números 🟡 Médio

Enunciado:

Crie uma função maior_de_tres(a, b, c) que receba três números e retorne o maior deles. Não use a função max() do Python — implemente a lógica você mesmo usando if. Teste com pelo menos 3 chamadas, incluindo números negativos.

Saída esperada:

maior_de_tres(4, 9, 2)    → 9
maior_de_tres(-1, -5, 0)  → 0
maior_de_tres(7, 7, 3)    → 7

Estrutura sugerida:

def maior_de_tres(a, b, c):
    maior = a
    if b > maior:
        maior = b
    # continue a lógica para c...
    return maior

print(maior_de_tres(4, 9, 2))
print(maior_de_tres(-1, -5, 0))
print(maior_de_tres(7, 7, 3))
💡 Dica

Comece assumindo que a é o maior. Depois compare com b: se b > maior, atualize maior = b. Repita para c.

Exercício 5 — Fatorial 🟡 Médio

Enunciado:

Crie uma função fatorial(n) que calcule o fatorial de um número inteiro não-negativo usando um laço for ou while. Lembre-se: 0! = 1 por definição matemática. Exiba o resultado para os números de 0 a 8.

Saída esperada:

0! = 1
1! = 1
2! = 2
3! = 6
4! = 24
5! = 120
6! = 720
7! = 5040
8! = 40320

Estrutura sugerida:

def fatorial(n):
    resultado = 1
    for i in range(1, n + 1):
        resultado = resultado * i
    return resultado

for num in range(9):
    print(f"{num}! = {fatorial(num)}")
💡 Dica

O fatorial é uma multiplicação acumulada: 5! = 5 × 4 × 3 × 2 × 1. Use uma variável resultado = 1 e multiplique por cada número do laço. O range(1, n+1) gera os números de 1 até n (inclusive).

Exercício 6 — Calculadora de média com conceito 🔴 Desafio

Enunciado:

Crie duas funções:

  1. calcular_media(notas) → recebe uma lista de notas e retorna a média aritmética.
  2. atribuir_conceito(media) → recebe a média e retorna o conceito conforme a tabela:
Média Conceito
>= 9.0 A
>= 7.0 B
>= 5.0 C
< 5.0 D

No programa principal, peça ao usuário para digitar 4 notas, armazene em uma lista, chame as duas funções e exiba o resultado final.

Saída esperada:

Digite a nota 1: 8.5
Digite a nota 2: 7.0
Digite a nota 3: 9.0
Digite a nota 4: 6.5

Notas: [8.5, 7.0, 9.0, 6.5]
Média: 7.8
Conceito: B

Estrutura sugerida:

def calcular_media(notas):
    soma = sum(notas)
    return soma / len(notas)

def atribuir_conceito(media):
    if media >= 9.0:
        return "A"
    elif media >= 7.0:
        return  # ?
    elif media >= 5.0:
        return  # ?
    else:
        return  # ?

# Programa principal:
notas = []
for i in range(4):
    n = float(input(f"Digite a nota {i + 1}: "))
    notas.append(n)

media = calcular_media(notas)
conceito = atribuir_conceito(media)

print(f"\nNotas: {notas}")
print(f"Média: {media:.1f}")
print(f"Conceito: {conceito}")
💡 Dica
  • sum(lista) soma todos os elementos de uma lista.
  • len(lista) retorna a quantidade de elementos.
  • Funções podem chamar outras funções! Você pode passar o resultado de calcular_media() direto para atribuir_conceito():
    conceito = atribuir_conceito(calcular_media(notas))
