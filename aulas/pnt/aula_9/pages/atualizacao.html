Atualização Controle de Empregados

  • Organizar o código javascript: (scripts.js)

    Criar as funções:
    • exibir() - Deve exibir a tabela com os dados dos empregados.
    • detalhar(i) - Exibe em um modal todos os dados de um empregado, baseado no seu índice.
    • editar(i) - Exibe um modal com caixas de textos com as informações do empregado, baseado no seu índice.
    • atualizar() - Deve atualizar os dados de um determinado empregado de acordo com os dados do modal, ao clicar no botão salvar.
    • excluir(i) - Deve excluir um determinado empregado do array, , baseado no seu índice. Utilize a função splice() para a exclusão.
    • totalEmpregados() - Deve retornar a quantidade de empregados cadastrados e mostrar abaixo da tabela, numa <div class="alert alert-info"></div>
    • pesquisar(nome_pesquisado) - Inserir um input text, e um botão para realizar pesquisa e retornar um novo array apenas nomes de empregados que tenham parte do texto como no enviado.
