Gere os Scripts com as respostas para os algoritmos a seguir:

Conteúdo base sobre arrays em python.

  1. Crie utilizando a função print() o seguinte menu:

    ===========================================
    == MENU DE CORES ==
    ===========================================
    == 1 - Cadastrar Cor
    == 2 - Remover Cor
    == 3 - Listar todas as Cores
    == 4 - Sair
    == Escolha uma opção
    ===========================================
  2. Crie um array vazio chamado cores.
  3. Coloque o menu dentro de um laço while, que se repita até a opção 4 for selecionada.
  4. De acordo com a opção escolhida realize as opções:
    1 - Solicitar ao usuário uma cor e adiciona-la ao array cores.
    2 - Solicitar o nome de uma cor e removela-la do array cores.
    3 - Listar todos os itens listados no array cores.
    4 - Sair do menu e apresentar a mensagem: Programa encerrado com sucesso.
