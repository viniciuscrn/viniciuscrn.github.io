1. Algoritmo para imprimir uma mensagem na tela:

mensagem = "Olá, mundo!"
print(mensagem)

2. Algoritmo para somar dois números:

a = 5
b = 3
soma = a + b
print(f"A soma de {a} e {b} é {soma}.")

3. Algoritmo para calcular a área de um retângulo:

altura = 5
largura = 3
area = altura * largura
print(f"A área do retângulo é {area}.")

4. Algoritmo para converter graus Celsius em Fahrenheit:

celsius = 30
fahrenheit = (celsius * 9/5) + 32
print(f"{celsius} graus Celsius equivalem a {fahrenheit} graus Fahrenheit.")
