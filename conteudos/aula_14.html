Gere os Scripts com as respostas para os algoritmos a seguir:

Conteúdo base sobre arrays em python.

  1. Crie um algoritmo onde o usuário informe um número e então seja exibido o resultado do fatorial deste número.
  2. A partir do array de números inteiros: [5, 6, 8, 9, 13, 3, 18, 21, 26, 31, 4, 8], crie um algoritmo que informe:
    • O maior valor do array.
    • O menor valor do array.
    • O maior valor do par.
    • O menor valor do ímpar.
  3. Crie um algoritmo onde o usuário informe uma palavra ou texto, e depois escolha uma letra. Logo após exibir como resultado a quantidade de vezes que a letra informada aparece no texto. Se a letra não existir informa com a mensagem: Letra inexistente no texto.
    Ex.:
    Informe um texto: Palmeiras maior que Machester City!
    Informe uma letra: i
    A letra i aparece 3 vezes no texto.
  4. Escreva um algoritmo onde o usuário informe um número, e seja exibido na tela todos os números antecessores a ele múltiplos de 2.
  5. Crie um array com 7 palavras e exiba apenas as palavras que possuem mais de 5 caracteres.
