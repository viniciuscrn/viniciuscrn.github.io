Gere os Scripts com as respostas para os algoritmos a seguir:

Conteúdo base sobre funções em python.

  1. Crie uma função que verifique se um número é par. A função deve receber um número como parâmetro e retornar True se o número for par e False caso contrário.
  2. Escreva uma função em Python que receba dois números como parâmetros e retorne a soma deles.
  3. Implemente uma função que receba uma string como parâmetro e retorne a quantidade de vogais presentes nessa string.
  4. Crie uma função que receba uma lista de números como parâmetro e retorne a média aritmética dos elementos dessa lista.
  5. Escreva uma função que receba uma lista de palavras como parâmetro e retorne a quantidade de palavras que possuem mais de cinco caracteres.
  6. Implemente uma função que receba uma lista de números como parâmetro e retorne uma nova lista contendo apenas os números pares presentes na lista original.
